== Installation ==

<h2>This section describes how to install the plugin and get it working.</h2>

e.g.

<h4>First things first, thank you for choosing our Magical Posts Display plugin!</h4>
<h4>You can install the plugin one of two ways:</h4>
<h5 class="short-info">Install using FTP</h5>
<ul>
 	<li>Unzip the <strong>magical-posts-display.zip</strong> file locally to your machine</li>
 	<li>Connect to your FTP area using your preferred FTP package</li>
 	<li>Upload the <strong>magical-posts-display.zip</strong> folder that you extracted from the zip file to the plugin folder of your WordPress installation (<strong>wp-content -&gt; plugin</strong>)</li>
 	<li>Go to <strong>plugin </strong> and activate Magical Posts Display.</li>
</ul>
<h5 class="short-info">Install by WordPress (recommended)</h5>
<ul>
 	<li>Login to your WordPress admin area.</li>
 	<li>Navigate to <strong>plugin</strong> and click the <strong>Add New</strong> button at the top.</li>
 	<li>Click the <strong>Upload plugin</strong> button at the top.</li>
 	<li>Click the <strong>Choose File</strong> button and locate the <strong>magical-posts-display.zip</strong> file on your machine and click the <strong>Install Now</strong> button.</li>
 	<li>On the basis the plugin installs correctly, click the <strong>Activate</strong> link.</li>
</ul>

## Privacy Policy
Magical Posts Display – Elementor Advanced Posts widgets uses [Appsero](https://appsero.com) SDK to collect some telemetry data upon user's confirmation. This helps us to troubleshoot problems faster & make product improvements.

Appsero SDK **does not gather any data by default.** The SDK only starts gathering basic telemetry data **when a user allows it via the admin notice**. We collect the data to ensure a great user experience for all our users.

Integrating Appsero SDK **DOES NOT IMMEDIATELY** start gathering data, **without confirmation from users in any case.**

Learn more about how [Appsero collects and uses this data](https://appsero.com/privacy-policy/).


== Frequently Asked Questions  ==

= How Can I use Magical Posts Display Plugin?  =

 You can easily use the plugin by Elementor page builder editor. If you go add new or edit page then you will get all Elementor widgets for use.

= How do I display Pages and Custom Post Types using Magical Posts Display?  =

To display Pages and Custom Post Types, simply select the desired content type from the settings within the Elementor editor. You can then customize the display options just as you would for Posts.
 
= Can I customize the look and feel of my content displays with Magical Posts Display?  =

Yes, you can customize the look and feel of your content displays using the various Elementor widgets that come with the addon. You can change the layout, colors, fonts, and more to create a unique and visually appealing content display

= Can I display posts from a specific category with Magical Posts Display?  =
Yes, Magical Posts Display supports posts categories, allowing you to display posts from a specific category. You can use the Elementor widgets to filter posts based on categories and tags and create a more personalized user experience for your visitors
 
= Do I need any coding skills to use Magical Posts Display?  =

 No, you don't need any coding skills to use Magical Posts Display. The addon comes with various Elementor widgets that allow you to create stunning content displays without any coding knowledge. Simply select the widget you want to use, customize it to your liking, and publish your content display.
 
= Can I use Magical Posts Display on multiple websites?  =

 Yes, you can use Magical Posts Display on multiple websites with a single license. The addon offers a flexible licensing system that allows you to use it on an unlimited number of websites, making it an excellent choice for web developers and agencies who create websites for multiple clients.
